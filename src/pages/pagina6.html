<p>Imagen de gatito</p>

Es importante recordar que todos los elementos img deben incluir un atributo alt. El contenido del atributo alt es fundamental para mejorar la accesibilidad, ya que es utilizado por lectores de pantalla y se muestra en caso de que la imagen no se cargue correctamente. Por ejemplo, un código como <img src="cat.jpg" alt="Es un gatito"> posee un atributo alt con el texto Es un gatito.
Dentro del elemento img, añade un atributo alt con este texto.
